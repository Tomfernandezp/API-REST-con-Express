API-REST creada con Express (Node.JS), utilizando base de datos MySQL y aplicacando el pátron MVC.

Se utilizó los siguientes conceptos para la realización de la API:
  -Patrón MVC
  -Node.JS
  -MySQL con buenas practicas
  -Solución de CORS

  API que se encarga de la alta, baja, modificación y obtención de ciertas peliculas.
